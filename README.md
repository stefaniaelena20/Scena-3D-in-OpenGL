  Acest proiect are ca scop dezvoltarea unei aplicatii grafice tridimensionale complete, utilizand
limbajul C++ si API-ul OpenGL. Proiectul urmareste demonstrarea competentelor in utlizarea
pipeline-ului grafic programabil,manipularea matriciala si implementarea algoritmilor de iluminare
si simulare fizica,simplificata.
  Obiectivul principal este crearea unei scene interactive care combina un mediu virtual
detaliat,elemente dinamice si efecte de medxiu.
